# Task Master

A full-stack web application with Next.js frontend and Node.js backend.

## Project Structure

```
├── back/           # Node.js Backend API
│   ├── src/        # Source code
│   ├── package.json
│   └── README.md
└── front/          # Next.js Frontend
    ├── src/        # Source code
    ├── package.json
    └── README.md
```

## Tech Stack

### Backend
- Node.js + Express.js
- PostgreSQL
- JWT Authentication
- Email verification

### Frontend  
- Next.js 15
- React 19
- TypeScript
- Styled Components
- i18next for internationalization

## Getting Started

### Backend Setup
```bash
cd back
npm install
npm run dev
```

### Frontend Setup
```bash
cd front
npm install
npm run dev
```

## Features

- User authentication and registration
- Email verification
- Modern responsive UI
- Multi-language support
- PostgreSQL database integration
